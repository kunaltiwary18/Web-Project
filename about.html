About Me

Anime Gallery

The Anime Gallery Website project is a simple yet engaging web application designed for a college project. This website is dedicated to collating and showcasing a collection of anime-related images, catering to anime enthusiasts and fans. The goal of this project is to create an accessible and visually appealing platform for sharing and enjoying anime art and images.
